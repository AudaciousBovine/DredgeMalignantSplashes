# Malignant Splashes
### Dark Splashes are more of a hazard now!
![Malignant Splashes](/MalignantSplash.png)


## Features
- Dark Splashes can no longer be discarded
- Have to be properly cleaned and removed at a shipyard (Sell for $0)
- OR
- Used in an Iron Rig recipe
- Morrow Shipyard now buys Materials
